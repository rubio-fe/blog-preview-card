# Blog Preview Card Solution

A solution to the Blog Preview Card challenge from Frontend Mentor.

## Overview

This project is a responsive blog preview card built with semantic HTML and modern CSS.  
The goal was to match the design as closely as possible while keeping the code clean and maintainable.

## The Challenge

Users should be able to:

- View the card layout on different screen sizes
- See hover states on interactive elements

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Variables
- Mobile-first workflow

## Features

- Clean semantic structure
- Responsive layout
- Pixel-accurate spacing and styling
- Custom properties for colors
- Hover interaction on the title
- Sharp shadow design (no blur)
